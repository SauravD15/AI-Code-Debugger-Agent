# AI Code Debugger

An intelligent AI-powered code debugging assistant designed to analyze, comment, and fix your code in real-time. Built for developers who want faster, smarter, and more insightful code reviews using cutting-edge AI models.

## Features

-  **Automatic Code Debugging** ‚Äì Instantly detects and suggests fixes for common bugs and logic errors.
-  **AI-Powered Comments** ‚Äì Annotates your code with meaningful comments for better understanding.
-  **Multi-language Support** ‚Äì Works with major programming languages (auto-detect enabled).
-  **Smart Suggestions** ‚Äì Offers best practices and optimization tips.
-  **Code Execution Sandbox** (coming soon)
-  **Theme Customization & UI Modes** ‚Äì Light/Dark modes for a better developer experience.
-  **GitHub Integration** ‚Äì Pull code directly from your repositories.
-  **Mobile-Responsive UI** ‚Äì Debug your code from any device.


##  Tech Stack

- **Frontend**: React, TailwindCSS
- **Backend**: Node.js, Express
- **AI Engine**: Google Gemini API
- **Database**: MongoDB / Firebase (Optional)
- **Others**: GitHub API, Prism.js, CodeMirror
